# Pakistan-Temporal-Network-Analysis-CPI-

This Project Deals with the Temporal analysis network of pakistan, How the prices of each city is interconnected with prices of another city. It takes cpi raw data as csv, passes through multiple data cleaning pipeline and then apply multiple sub working to determine how each city is connected with other. This Project highly emphasises on the concept of nodes connection on basis of temporal connections using cosine similarity.
It will calculate and interpret 4 different measures of centrality, namely degree, closeness, betweenness, and eigenvector centrality, in order to identify the cities that act as economic hubs within each category.It has been implemented and compared with several different weighting strategies for the aggregation of centrality measurements: equal weighting, correlation based weighting, entropy based weighting and interactive and user defined weighting strategies.
